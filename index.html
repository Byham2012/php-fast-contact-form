Logo Style Guide

Wordmark A logo that consists only of the brand name in a custom font (e.g., Google, Coca-Cola).

Lettermark A logo made from the brand's initials, useful for long business names (e.g., IBM, HBO).

Pictorial Mark A logo that features a recognizable icon or symbol (e.g., Apple, Twitter).

Abstract Mark A unique geometric or abstract symbol that represents the brand (e.g., Nike Swoosh, Adidas).

Mascot A logo that features a character or brand mascot (e.g., KFCâ€™s Colonel Sanders, Pringles).

Emblem A logo where text is inside a symbol, often looking like a badge (e.g., Harley-Davidson, Starbucks).
